# 🎭 Fallas València 2026

Progressive Web App interactiva para explorar las Fallas de València 2026.

## ✨ Características

- 🗺️ Mapa interactivo con 702 fallas
- 📅 Calendario de eventos (22 Feb - 19 Mar)
- 🔍 Búsqueda inteligente con scoring
- 🛣️ Sistema de rutas optimizadas
- 🧭 Navegación en tiempo real
- 🎆 Animaciones de eventos
- 📱 Mobile-first design
- 🎨 Diseño Wander Guide (glassmorphism)

## 🚀 Quick Start

```bash
# Instalar dependencias
npm install

# Configurar Mapbox token
cp .env.example .env
# Edita .env y añade tu token de Mapbox

# Desarrollo
npm run dev

# Build
npm run build

# Deploy a GitHub Pages
npm run deploy
```

## 📦 Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Mapbox GL JS
- Zustand
- Lucide React

## 🎨 Diseño

Basado en Wander Guide:
- Colores cálidos (naranja coral + amarillo)
- Glassmorphism en overlays
- Bottom sheets para modales
- Tipografía: DM Sans + Space Grotesk
